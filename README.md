# Khari Ui Design System

A design system created with storybook where you can find tokens and components

## Functionalities

- Colors
- Font Sizes
- Font Weights
- Font Families
- Line Heights
- Radii
- Space
- Avatar
- Toast
- Tooltip
- Box
- Button
- Checkbox
- Multi Step
- Text Area
- Text Input
- Heading
- Text
